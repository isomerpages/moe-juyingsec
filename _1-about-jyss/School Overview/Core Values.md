<h4><strong>The JYian Code of Conduct</strong></h4>
<p>The JYian Code of Conduct (CoC) guides students on conduct that exemplifies
the school values of&nbsp;&nbsp;R2IGHT.&nbsp; Whilst the CoC provides the
standards, the school rules set the boundaries of acceptable behaviour.</p>
<p>As a member of the JYian Community, I shall do my utmost to uphold the
school values.</p>
